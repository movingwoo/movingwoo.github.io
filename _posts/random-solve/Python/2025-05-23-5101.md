> #### 설계  
---  
  
음 쉬운 문제  
하루의 마무리로 딱이다.  
  
시작 숫자와 차이가 주어지고 다음 숫자가 수열의 항인지 판단하는 문제이다.  
  
> #### 구현  
---  
  
단순 덧셈 반복해도 문제없지만 조금 더 스마트하게 해봐야지.  
  
```python
while True:
    a, b, c = map(int, input().split());

    if a == 0 and b == 0 and c == 0:
        break

    # b가 0이면 예외처리
    if b == 0:
        if a == c:
            print(1)
        else:
            print("X")
    # 음수 고려
    elif (c - a) % b == 0 and (c - a) // b >= 0:
        print((c - a) // b + 1)
    else:
        print("X")
```
